Resource Profile: CL Receta

Official URL: https://rme.hl7chile.cl/StructureDefinition/RecetaCl Version: 0.9
Active as of 2022-12-20 Computable Name: MedicamentoRecetaCL

Este Perfil utiliza este recurso como un contenedor de Prescripciones, de tal forma de poder ser utilizado como formato de Receta electrónica con los fármacos indicados por el clínico durante un acto de indicación farmacológica.

Usage:

  • Examples for this Resource Profile: RequestGroup/Ejemplo1-Receta, RequestGroup/Ejemplo2-Receta, RequestGroup/Ejemplo3-Receta, RequestGroup/Ejemplo4-Receta, RequestGroup/Ejemplo5-Receta, RequestGroup/Ejemplo6-Receta, RequestGroup/Ejemplo7-Receta
